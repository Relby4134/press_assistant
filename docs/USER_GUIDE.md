# Руководство лектора — Ассистент лектора

## Обзор

«Ассистент лектора» — надстройка PowerPoint, которая транслирует слайды студентам
через Telegram и позволяет собирать вопросы прямо во время лекции.

**Как это работает:**
1. Лектор запускает лекцию в панели надстройки
2. Студенты подключаются к боту и сканируют QR-код
3. При смене слайда изображение автоматически уходит всем студентам в Telegram
4. Студенты задают вопросы в боте — лектор видит их в панели

---

## Интерфейс лаунчера

Лаунчер — это приложение, которое запускает сервер и управляет настройками.

### Вкладка «Управление»

| Элемент | Назначение |
|---------|-----------|
| Индикатор статуса | Зелёный — сервер работает, серый — остановлен, красный — ошибка |
| **▶ Запустить** | Запустить сервер |
| **■ Остановить** | Остановить сервер |
| **📂 Выбрать файл...** | Открыть презентацию в PowerPoint (активна только при запущенном сервере) |

### Вкладка «Бот»

- **Токен бота** — токен от @BotFather. Нажмите 👁 чтобы показать/скрыть
- **Имя бота** — @username вашего бота
- **Доступные команды** — выберите какие команды бота активны:
  - `/join` — подключение студентов к лекции по названию
  - `/slide` — запрос текущего слайда
  - Вопросы — приём текстовых сообщений как вопросов

### Вкладка «Установка»

Используется при первичной настройке. Подробнее — в [руководстве по установке](INSTALLATION.md).

## Проведение лекции

### 1. Запуск сервера

Откройте лаунчер и нажмите **▶ Запустить**.
Дождитесь зелёного статуса — это занимает 5–15 секунд.

### 2. Открытие презентации

В лаунчере нажмите **📂 Выбрать файл...** или откройте `.pptx` файл обычным способом.

### 3. Открытие панели надстройки

В PowerPoint: вкладка **Главная → Ассистент → Открыть панель**.

### 4. Начало лекции

В панели надстройки:

1. Введите **название лекции** (например: «Лекция 3 — Алгоритмы сортировки»)
2. Выберите **режим участия**:
   - **Анонимный** — студенты подключаются без имени
   - **Поимённый** — студенты вводят своё Ф.И.О. при подключении
3. Нажмите **Начать лекцию**

После запуска откроется вкладка **QR-код** — покажите его студентам для подключения.

### 5. Трансляция слайдов

**Автоматически:** при переходе на новый слайд студентам уходит **уведомление** с номером слайда.
Студент может запросить изображение сам — кнопкой в боте или командой `/slide`.
Статус «Слайд N отправлен ✓» подтверждает успешную отправку уведомления.

**Вручную:** отключите переключатель «Отправлять слайд автоматически» —
появится кнопка **📤 Отправить слайд студентам**.

### 6. Работа с вопросами

Перейдите на вкладку **Вопросы**. Новые вопросы отображаются сверху.

- Число в значке вкладки — количество непрочитанных вопросов
- Кнопка **↻** — обновить список вручную (обновление идёт каждые 10 секунд автоматически)
- Кнопка **✕** рядом с вопросом — удалить вопрос из списка

### 7. Рассылка сообщения

Введите текст в поле внизу и нажмите **➤** или Enter.
Сообщение уйдёт всем подключённым студентам в Telegram.

### 8. Список студентов

На вкладке **Студенты** отображаются все подключённые участники.
Кнопка **Выгнать** — отключить студента от лекции.

### 9. Аналитика

На вкладке **Аналитика** — кто и сколько раз запрашивал слайды.

### 10. Завершение лекции

Нажмите кнопку **✕ Завершить** в верхней части панели.
Все студенты получат уведомление об окончании лекции.

## Режим показа слайдов (SlideShow)

При запуске режима показа (`F5`) надстройка переходит в режим опроса:
каждые 800 мс проверяет текущий слайд и отправляет его при смене.

> В режиме показа надстройку не видно!
> Решение проблемы: показывать слайды в режиме чтения, а окно надстройки выносить на другой экран.

## Советы

- Название лекции должно быть уникальным на каждом занятии, чтобы студенты
  не попали на прошлую сессию
- QR-код ведёт напрямую в Telegram-бот с предзаполненной командой подключения —
  студенту достаточно нажать «Старт»
- Поимённый режим удобен для контроля посещаемости
