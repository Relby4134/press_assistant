# Руководство по установке — Ассистент лектора

## Системные требования

| Компонент | Требование |
|-----------|-----------|
| ОС | Windows 10 / 11 (64-bit) |
| PowerPoint | Microsoft Office 2016 или новее |
| Интернет | Нужен для Telegram Bot API |
| Права | Администратор локального компьютера |

Java устанавливать **не нужно** — она встроена в установщик.

---

## Шаг 1. Установка приложения

1. Запустите `PresAssistant-1.0.0.msi`
2. Следуйте мастеру установки: выберите папку, нажмите **Установить**
3. После завершения на рабочем столе появится ярлык **PresAssistant**

---

## Шаг 2. Запуск и настройка

Откройте **PresAssistant** через ярлык на рабочем столе или меню Пуск.

### Вкладка «Бот»

Заполните перед первым запуском:

- **Токен бота** — получите у @BotFather в Telegram командой `/newbot`
- **Имя бота (@username)** — username вашего бота, например `@my_lecture_bot`

### Вкладка «Управление»

Нажмите **▶ Запустить** — сервер запустится, статус сменится на зелёный.

---

## Шаг 3. Установка SSL-сертификата

Перейдите на вкладку **«Установка»** и нажмите **Установить...** рядом с «Шаг 1 — SSL-сертификат».

- Если появится окно UAC — нажмите **Да**
- После успешной установки статус сменится на ✔

> Сертификат нужен для защищённого соединения между PowerPoint и сервером.
> Устанавливается один раз.

---

## Шаг 4. Регистрация надстройки

На вкладке **«Установка»** нажмите **Зарегистрировать...** рядом с «Шаг 2».

> Сервер должен быть запущен (шаг 2) перед регистрацией.

После регистрации:

1. Полностью закройте PowerPoint
2. Убедитесь через Диспетчер задач что процесса `POWERPNT.EXE` нет
3. Откройте PowerPoint снова

---

## Шаг 5. Подключение надстройки в PowerPoint

1. Откройте любую презентацию `.pptx`
2. Перейдите на вкладку **Вставка → Мои надстройки**
3. Выберите вкладку **Общая папка**
4. Нажмите **Ассистент лектора → Добавить**

Кнопка **Открыть панель** появится на вкладке **Главная**.

---

## Решение проблем

### Кнопка «Установить сертификат» ничего не делает

На этом компьютере отключён UAC. Это нормально — скрипт всё равно запустится,
если у вас есть права администратора.

### Надстройка не появляется в «Мои надстройки»

- Убедитесь что нажали «Зарегистрировать надстройку» при запущенном сервере
- Полностью перезапустите PowerPoint (проверьте Диспетчер задач)

### После открытия надстройки — «Содержимое заблокировано»

- Убедитесь что сертификат установлен (✔ в Шаге 1)
- Убедитесь что сервер запущен (зелёный статус на вкладке «Управление»)
- Перезапустите PowerPoint

### Сервер не запускается

- Проверьте что токен бота введён правильно
- Порт 8082 должен быть свободен. Проверить: `netstat -an | findstr 8082`

---

## Удаление

Перейдите в **Параметры → Приложения**, найдите **PresAssistant** и нажмите **Удалить**.

Данные лекций хранятся в `%USERPROFILE%\.presassistant\` — удалите вручную при необходимости.
