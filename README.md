# Ассистент лектора (PresAssistant)

Система для интерактивного сопровождения лекций: транслирует слайды PowerPoint
студентам через Telegram и собирает вопросы в реальном времени.

---

## Как это работает

1. Лектор запускает **PresAssistant** и открывает презентацию в PowerPoint
2. В панели надстройки вводит название лекции и нажимает «Начать»
3. Студенты сканируют QR-код и подключаются к Telegram-боту
4. При каждой смене слайда оповещение автоматически уходит всем студентам
5. Студенты запрашивают слайд, если им необходимо
6. Студенты задают вопросы в боте — лектор видит их в панели надстройки

## Быстрый старт

### Требования
- Windows 10/11 (64-bit)
- Microsoft Office 2016 или новее
- Telegram-бот (создаётся через @BotFather)

### Установка
1. Скачайте и запустите `PresAssistant-1.0.0.msi`
2. Откройте лаунчер, введите токен бота на вкладке «Бот»
3. Запустите сервер → установите сертификат → зарегистрируйте надстройку
4. Перезапустите PowerPoint

Подробнее: [docs/INSTALLATION.md](docs/INSTALLATION.md)

## Стек технологий

| Слой | Технологии |
|------|-----------|
| Сервер | Java 21, Spring Boot 4, H2, Flyway, Telegram Bots API |
| Надстройка | Office Add-in JS, webpack 5, QRCode.js |
| Лаунчер | Java Swing, FlatLaf, jpackage → MSI |
| Безопасность | HTTPS (самоподписанный сертификат), WiX 3.11 |
